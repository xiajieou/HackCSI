# HackCSI

A modern hackathon website for the City University of New York, College of Staten Island.

## 🚀 Features

- **Modern Design** - Blue color theme with animated backgrounds and smooth scrolling
- **Responsive** - Fully responsive design for all screen sizes
- **Accessible** - ARIA labels, skip-to-content, and keyboard navigation
- **SEO Optimized** - Meta tags, Open Graph, sitemap, and robots.txt
- **PWA Ready** - Web app manifest included

## 📋 Sections

- Hero with event stats
- About with challenge tracks
- 2-day event schedule
- FAQ with accordion
- Tiered sponsors section
- Registration form
- Footer with social links

## 🛠️ Tech Stack

- [Next.js 16](https://nextjs.org/) - React framework
- [TypeScript](https://www.typescriptlang.org/) - Type safety
- [Tailwind CSS](https://tailwindcss.com/) - Styling
- [shadcn/ui](https://ui.shadcn.com/) - UI components

## 🏁 Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

## 🌐 Deployment

This project is optimized for [Vercel](https://vercel.com) deployment:

```bash
npx vercel
```

Or connect your GitHub repository to Vercel for automatic deployments.
